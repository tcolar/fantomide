How Do I connect to database?
Open Settings.fan file in your application's conf directory. Set values for "db-host-string", "db-username", "db-password".
What Next?
Start by modifying the Index.html and Index.fan Files. You can define more routes in the Routes.fan file in your application's "conf" directory.
